# LicensePlate Module Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `LicensePlate` module (validate/normalize/format/parse + special-plate classification) for AT, DE, CH, HR, TR to BOTH the Dart package and the TypeScript package `@kreiseck/validator`, proven by shared JSON vectors.

**Architecture:** One module per package (`lib/src/license_plate/`, `js/src/license-plate/`). Per-country grammar + classification is hand-written; per-country `code â†’ region` tables are curated from public-domain/official sources by a new generator that emits Dart (`plate_metadata.g.dart`) and JSON (`js/src/data/plate-metadata.json`) in one run. Parity enforced by `test/vectors/license_plate.json`.

**Tech Stack:** Dart + TypeScript (mirrored logic), Python 3 generator (curated dicts + a committed KBA-derived CSV for DE).

## Global Constraints

- **Two packages, same behaviour.** Every country stage lands in Dart AND TypeScript and passes the same `license_plate.json` vectors in both.
- **Zero runtime dependencies** in both published packages. Data is bundled (Dart const maps / TS JSON); no runtime fetch.
- **No AI/tool attribution** anywhere â€” code, comments, data, docs, commit messages, package metadata. The generated-file header (`Generated by tool/gen_plate_metadata.py`) is the only allowed provenance line.
- **Region data is public-domain / official-sourced** (KBA amtliches Werk for DE; official canton/province/Bezirk lists for the rest). NO ODbL/openpotato data.
- **API parity:** `isValid/validate/normalize/format/tryFormat/parse`, each taking an optional `{ country }` (ISO2 string). `normalize`/`format` throw on invalid; `tryFormat` returns null.
- **Classification is best-effort:** when a special-plate rule is uncertain, `type` is `standard`. Classification never changes the valid/invalid outcome.
- **New IssueCodes** (identical in Dart enum and TS union): `plateEmpty`, `plateBadChars`, `plateBadFormat`, `plateUnknownCountry`, `plateAmbiguousCountry`.
- Module name is `LicensePlate`; TS subpath `@kreiseck/validator/license-plate`.
